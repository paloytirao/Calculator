# Retro Calculator

A simple retro-style calculator inspired by the provided design, built with HTML, CSS, and vanilla JavaScript. It features a dark theme, green LCD-like display, and basic arithmetic operations.

## Demo

Open `index.html` in any modern web browser to use the calculator.

## Features

- **Basic Operations**: Addition (+), subtraction (-), multiplication (*), and division (/).
- **Decimal Support**: Add decimal points to numbers.
- **00 Button**: Quick input for double zeros.
- **Clear Function**: Reset with the 'C' button.
- **Error Handling**: Displays 'Error' for division by zero.
- **Responsive Design**: Adapts to smaller screens.
- **No Dependencies**: Pure vanilla JS, no libraries required.

## Installation

1. Clone or download the files (index.html, script.js, styles.css).
2. Open `index.html` in your browser.

No setup or dependencies needed.

## Usage

- Click numbers to input digits (including 00 for faster entry).
- Click operators to perform calculations.
- Click '=' to get the result.
- Click 'C' to reset.
- Supports chaining operations (e.g., 2 + 3 * 4 =).

Example: 5 + 3 = displays 8.

## Technologies Used

- **HTML5**: For structure.
- **CSS3**: For retro styling and layout (using CSS Grid).
- **JavaScript**: For calculation logic and event handling.

## Project Structure

- `index.html`: Main structure.
- `script.js`: Calculator logic.
- `styles.css`: Styling to match the retro design.

## Limitations

- No keyboard support in this version.
- No advanced functions (e.g., percentages, square roots).
- No expression preview (shows only current input/result).
- Basic error handling only for division by zero.
